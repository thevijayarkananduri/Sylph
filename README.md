# Sylph — Weather & Air

A beautiful weather and air quality app with personality.

## Features
- Live weather data from OpenWeatherMap
- Air Quality Index from WAQI
- Animated backgrounds that change with weather
- Outfit recommendations for "leaving now"
- Temperature unit toggle (°C / °F)
- Search history with local storage

## Built With
- Flutter
- OpenWeatherMap API
- WAQI API
